Chapter 1 Introduction

This is a summary of the Health Care Avoidance Suvery created by the Economic Evaluation Research Group and implemented in Qualtrics. There are several broad categories of variables inlcuded in the survey and variables will be summarized first by these categories.

For this project, variables are named using four parts. First a prefix indicating categories of variables in the survey, “demo” in this case to refer to demographic variables. The suffix of each variable indicates the variable type including bin for binary, cont for continuous, cat for categorical (unordered), ord for ordered categorical, and chr for characters or text. The second part of variable names, immediately following the suffix indicate the sub-category such as “age” or “ins” for age and health insurance-related variables, respectively. In some cases, to differentiate variables in a sub-category, a third part is included in the variable name, before the suffix.
